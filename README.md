# My Borrowed Books 📚

A sleek, modern, and lightweight web application designed to help you keep track of books you have borrowed from the library. With real-time countdowns, status indicators, and clean aesthetics, you'll never miss a return date again.

## ✨ Features

- 📌 **Log Borrowed Books**: Easily save the book title, author, genre, and return date.
- ⏳ **Loan Countdown & Progress**: Visual progress bars and countdown badges showing exactly how much time is left.
- 🎨 **Genre Spine Accents**: Dynamic color coding based on the book's genre, styled to mimic a real book spine.
- 🔍 **Real-Time Search**: Instantly filter books by title or author as you type.
- 🎛️ **Quick Status Filters**: Filter your collection by **All**, **Overdue**, **Due Soon**, and **Safe**.
- 🔔 **Interactive Toasts**: Clean micro-interactions and status alerts.
- 💾 **Local Storage Persistence**: Your library data is automatically saved in your browser and persists across reloads.
- 📱 **Fully Responsive**: Designed to look great on desktop, tablet, and mobile devices.

## 🚀 Getting Started

No installation or build steps are required. Simply open `index.html` in any modern web browser to run the application.

1. Clone the repository:
   ```bash
   git clone https://github.com/RanjithKumarM-BEcse/My_Books.git
   ```
2. Navigate into the project directory:
   ```bash
   cd My_Books
   ```
3. Open `index.html` in your browser.
